# Intel Dashboard

React CRM dashboard for the [Intel](https://github.com/TheMitchyBoy/Intel) newspaper intelligence API.

In production, the built app is served from the API at port 8000. Use this package for local frontend development only.

```bash
npm install
npm run dev     # http://localhost:5173
npm run build   # output → dist/ (copied to ../static in Docker)
```

See the root [README](../README.md) and [CONTRIBUTING](../CONTRIBUTING.md) for full setup.
